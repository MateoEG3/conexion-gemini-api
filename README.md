# Conexión a la API de Gemini con Python

## Descripción

Este proyecto implementa una conexión a la API de Gemini utilizando Python. El programa realiza una consulta al modelo Gemini y muestra la respuesta generada en la terminal.

## Requisitos

* Python 3.12 o superior
* API Key de Gemini
* Conexión a Internet

## Instalación

### Crear entorno virtual

```bash
python -m venv venv
```

### Activar entorno virtual

```powershell
.\venv\Scripts\Activate.ps1
```

### Instalar dependencias

```bash
pip install -r requirements.txt
```

## Configuración

Crear un archivo `.env` con el siguiente contenido:

```env
GEMINI_API_KEY=TU_API_KEY
```

## Ejecución

```bash
python app_gemini.py
```
